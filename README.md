Aplicación para el registro de libros en una biblioteca
La solución planteada tendrá las siguientes características:
• Reportería: visualizar, consultar, los elementos que conforman la estructura de datos
que haya seleccionado.
• Agente de IA utilizado: En el caso del uso de un agente de IA, mencione y describa
cual fue el agente, el porcentaje de código escrito usando el agente.
